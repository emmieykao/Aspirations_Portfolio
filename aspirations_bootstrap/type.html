Basics

Type

The type function simply returns the type of an object. Objects can have several different types, and here are some of the most common:
Type Abbreviation Explanation Examples
String str Can be anything but will not possess any special qualities. For instance, one cannot add '1' and '2' if they are both strings; they must be converted to integers. 'Hello World', '1', '0.3'
Integer int Any whole number; can be added, multiplied, etc. in equations -5, 0, 100
Float float Any decimal; possesses the same characteristics as int -6.12, 0.324, 3.1415
List list A group of objects contained in square brackets and separated by commas ['Hello', 9, 0.1, True]
Dictionary dict Similar to a list but organizes elements in key:value pairs; also uses curly brackets. {'First':'Bob', 'Last':'Smith', 'Age':35}
You can simply find the type of an object by using type():
#Input:
print(type('Hello'))
#Output:
<class 'str'>
Or:
#Input:
print(type(90))
#Output:
<class 'int'>
Or:
#Input:
print(type(99.99))
#Output:
<class 'float'>

Type functions are especially helpful when debugging, since your computer often expects one type of variable but recieves another!
Section Challenge:
Get the user to input a character, and then test the type() function on their input. What do you notice?
user_input = input("Enter a character: ") print(type(user_input))
Inputs will always appear as strings, and you will have to convert them in order to make them into other types.

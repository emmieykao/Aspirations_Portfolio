Basics

Input

The input function is relatively similar to the print function, but it asks the user for an answer:
#Input:
input("What is your favorite color: ")
#Output:
What is your favorite color: Blue
Try it yourself! You can write anything in an input function:
You can also store a user's input as a variable, then use it later:
#Input:
userName = input("What is your name: ")
print(f"Hello, {userName}!")
#Output:

What is your name: Bob
Hello, Bob!

Make sure to account for any input a reader could put into your code. For instance, if you were expecting a number but recieve a word instead, make sure your code knows what to do with it!

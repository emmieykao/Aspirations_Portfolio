For Loops

Finally, for loops are the best way to complete repeated tasks with few lines of code:
#Input:
for i in range(5): 
    print(i)
#Output:
1
2
3
4
5
For loops can be used to loop through different values. You can also use if statements when writing for loops:
#Input:
for i in range(15): 
    if i % 5 == 0:
        print(i)
#Output:
5
10
15
Make sure to account for any input a reader could put into your code. For instance, if you were expecting a number but recieve a word instead, make sure your code knows what to do with it!

Separate Python Libraries

Here is a list of several python libraries as well as links and descriptions for further exploration:

math

  • Can execute more complicated mathematical calculations

  • Can get exact values for irrational numbers like 𝜋

pprint

  • Means "Pretty Print"
  • Can format otherwise hard-to-read objects
  • Can let you format a complicated object in specific way (margins, etc)

numpy

  • Multifunctional python library that draws on languages like C
  • Can create n-dimensional arrays with indexing and vectorization
  • Works best alongside other data science-based libraries such as pandas

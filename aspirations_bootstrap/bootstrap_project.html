Introduction

Python is a user-friendly programming language that uses built-in functions and libraries in order to complete a variety of tasks. In fact, Python is not only built to create any one task. The Python official website states that:

Python's simple, easy to learn syntax emphasizes readability and therefore reduces the cost of program maintenance. Python supports modules and packages, which encourages program modularity and code reuse. The Python interpreter and the extensive standard library are available in source or binary form without charge for all major platforms, and can be freely distributed.
